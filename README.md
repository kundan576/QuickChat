# QuickChat
WhatsApp-style CRUD chat app using Express &amp; MongoDB

QuickChat is a WhatsApp-style CRUD chat application built using
Node.js, Express.js, MongoDB, and Mongoose.

## Features
- Create chats
- Read messages
- Update messages
- Delete messages
- RESTful routing

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS

## Run Locally
npm install  
node index.js  

Open: http://localhost:8080/chats
